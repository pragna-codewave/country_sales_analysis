# Global Sales Data Analysis (SQL Project)

## Project Overview
This project performs end-to-end SQL analysis on multi-country sales data.

It focuses on:
- Data integration from multiple countries
- Data cleaning and transformation
- Business insights and performance analysis

## Business Objectives
- Identify top-performing countries
- Analyze product and sales performance
- Understand customer behavior
- Evaluate profit margins
- Track sales trends over time

## Tools Used
- SQL Server
- Excel for cleaning data
- Power BI for dashboard

## Key Analysis

### Revenue & Profit by Country
Identifies countries generating highest revenue and profit.

### Top Products
Finds best-selling products based on quantity.

### Sales Performance
Identifies top-performing sales representatives.

### Store Analysis
Evaluates performance of store locations.

### Monthly Trends
Tracks revenue trends over time.

### Profit Margin Analysis
Compares profitability across countries.


## Key Insights
- Revenue is concentrated in a few countries
- Certain products dominate sales volume
- Profit margins vary across regions
- Customer demographics influence revenue
- Payment methods impact transaction volume
